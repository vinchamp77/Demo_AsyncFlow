# Asynchronous Flow Demo App
Simple Android app to demonstrate asynchronous flow in Kotlin & Jetpack Compose

## Requirements
- Android Studio Dolphin or later

## Tech Stack
- Jetpack Compose
- LiveData
- Flow and Channel
- StateFlow and SharedFlow
